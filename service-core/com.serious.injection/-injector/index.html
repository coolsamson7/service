Injector

class Injector : BeanPostProcessor

An Injector is responsible for the complete dependency injection of target objects. Basically, it performs two tasks:

  • It serves as a registry for Injections. An injector is registered for a certain type of annotation. the injection manager can only process annotated fields and methods if there is an injector registered for the annotation type.

  • It performs the injection of resources into a target object. Therefore, it computes all annotations of fields and methods of the target type, looks up registered injectors for the annotations' types and - if present - calls, one after the other, the inject method of these injectors for the target object.
