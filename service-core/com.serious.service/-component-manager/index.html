ComponentManager

@Component
class ComponentManager : ApplicationContextAware

The ComponentLocator has already scanned and registered the corresponding descriptor and beans. In a postConstruct phase the following logic is executed

  • component descriptors are copied fom the locator

  • local implementations are instantiated

  • local components are published to the registry

  • local components start up
